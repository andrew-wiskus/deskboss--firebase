COMMANDS

  1. example task #4 -> adds task that has a scrum count of 4
  2. $dir dirName -> changes into directory: dirName, if not exists creates new.
  3. $dir .. -> goes to parent directory
  4. ~ -> puts you back in top folder, '/main'
  5. $delete main/folder/folder.. -> deletes folder/children folder and all tasks within
  6. $bug bugText #4-> adds bug to global bug db for all users to see. Priority added via #
  7. $friend add friend@email.com ->send friend request
  8. $pojo create pojoName -> WIP
  9. $pojo invite member@email.com -> WIP
